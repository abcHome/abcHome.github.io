成本核算

成本对比

买车唯一的替代方案是租车。 租车每小时大概10元起步。 做最保守的假设， 每周买菜一次（耗费1小时），出去吃饭一次（耗费2小时）。算上还车，路途等待等我们额外补偿一小时，也就是每周要40元， 一个月160元。 买车的话（普通代步车而言）， 基础保险+油钱大概120左右（基于个人情况， 基本代步车）。很明显：

  • 租车
    • 不必担心交通事故影响后期卖车
    • 活动受限制
  • 买车
    • 需要考虑日后卖车（代步车贬值不会太多）
    • 活动更加自由

粗略分析，如果能接受最保守的假设， 那买车并不是必须的。 套用朋友的一句话“买车后生活质量显著提高。就看最后卖车价格如何了。“但话说回来， 都在于个人的选择了， 哪有这么多瞻前顾后，重要的是物尽其用，给生活带来方便， 学有所成就值得！

买车开销

  • 开销类型
    • 给卖家的付款
    • 税+注册费
      • 税费根据listed price进行计算
    • 保险
      • 多多询价（实地去看， 网络quote）都要！代步车有个最基本的强制险就行了
    • 油钱
      • 代步车而已， 放心开（怎么都比国内舒服！）
      • 靠近ypslanti， 油价更低！
    • 保养费用（基础）
      • 基础养护也就是换个机油， 没必要4S店（dealer)
      • 换轮胎Costco， ArborCenter都可以

车辆来源比较

朋友

  • 优点：
    • 知根知底， 不需要太多顾虑。大家都很坦诚
    • 税费可以节约通过listed priced
  • 缺点：
    • ”极品“卖家or买家

      同学

  • 优点：
    • 汽车使用习惯良好
    • 见面方便
    • 很低的诈骗风险
  • 缺点：
    • 同上

      dealer

  • 优点：
    • 法律保障（新车）；
    • 养护方便（新车和pre-owned）
  • 缺点：
    • 提供的服务， 能以更低价格在别处获得
    • 业余的去找专业的， 业余的发现不了的毛病都是修不好的毛病
    • 提供的价格外，还要清偿额外的很多款项
  • 建议：
    • 西边的Jackson Rd 或者南边 S State都可以
    • Brairwood 旁边的dealer， bullshit！

      Craigslist（美国的58同城）

  • 优点：
    • 个人之间的交易
    • 物有所值
  • 缺点：
    • 需要懂车：车子很多部分的检查需要懂行人才行
    • 需要耐心：期待的车子或许还没有放到市场
  • 建议：
    • 多观摩，遇到too good to be real往往就是诈骗
    • 没有照片的， 照片精修的都往往是骗子
    • 地点在det的一般是翻新车
    • 找一个懂车的朋友去试车很重要！
    • 不要只看里程(mileage), 一辆天天高速上下班的车1000miles要好过天天堵车100miles的车；更好过”趴窝不动“的车

买车应用工具

通用性工具

通用性工具是不管通过任何途径， 都会用到的工具。

  • kbb是汽车估价网站， 朋友之间价格的界定就靠kbb了，这是市场给出的合理价格区间。至于具体的价格，就看双方的意愿。 同时，过分偏离kbb估值的二手车， 可能是捡漏， 也可能是骗局！！
  • VIN免费查找是用来初步查看二手车历史的网站。（注意！免费网站的结果不具有任何保证性， 但免费网站都报告汽车有问题， 这车必定有问题！）
    • 查看以下：
      • 事故为0吗？
      • clean title吗？
      • 养护记录（如果有）
        • 一辆养护良好的老车不逊于一个恣意使用的新车
        • 暴躁的驾驶对汽车有害（买车也是买车主）

          定制用途工具

  • cars是一个搜索引擎， 帮助快速确定选车区间， 但提供大部分为dealer库存的二手车， 但我们通过此能知道某年份的某车， 大概里程，价格等信息。
  • carfax (淘宝买！)
    • 当你有看上眼的车， 而且免费查没有问题
    • 请求一份carfax报告会给你上述所有信息， 付费的信息更有详实的历任车主记录， 在试车的时候更加有的放矢的提问。
